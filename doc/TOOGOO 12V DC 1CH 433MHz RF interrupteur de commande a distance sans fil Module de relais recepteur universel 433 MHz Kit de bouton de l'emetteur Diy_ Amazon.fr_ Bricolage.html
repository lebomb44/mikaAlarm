Achat vérifié
Produit 100% satisfaisant et impressionnant par ses fonctions!
En bidouillant un peu j'ai déterminé la procédure de codage:
- Peut mémoriser 25 codes (433.92 MHZ) Attention ne fonctionne qu'avec les télécommandes à code fixe, fonctionne également avec les télécommandes encastrés.
La procédure: appuyer 1 à 8 fois sur le bouton apprentissage, attendre la LED fixe, choisir le bouton de la télécommande p.ex. "A", la LED clignote 4 fois, le bouton A est mémorisé

Appuyer 1 x sur le bouton apprentissage ; appuyer "A": Mode impulsion appuyer "A", le relais est activé tant que le bouton "A" est appuyé.
Appuyer 2 x sur le bouton apprentissage; appuyer "A": Mode toggle (appuyer "A" pour activer & ré-appuyer "A" pour désactiver)
Appuyer 3 x sur le bouton apprentissage ; appuyer "A", la LED s'éteint, attendre 1 seconde la LED se ré-allume, appuyer "B" la LED clignote 4 fois = OK : "A" active le relais, "B" le relâche. (Ou une télécommande pour activer et une autre pour désactiver)
Appuyer 4 x sur le bouton apprentissage ; appuyer "A": Appuyer "A" Mode timer le relais reste activé pour 5 secondes
Appuyer 5 x sur le bouton apprentissage; appuyer "A": Appuyer "A" Mode timer le relais reste activé pour 10 secondes
Appuyer 6 x sur le bouton apprentissage; appuyer "A": Appuyer "A" Mode timer le relais reste activé pour 15 secondes
Appuyer 7 x sur le bouton apprentissage; appuyer "A": Appuyer "A" Mode timer le relais reste activé pour 20 secondes
Changer le condensateur devrait modifier ces délais.
Appuyer 8 x sur le bouton apprentissage: Remise à zéro de toutes les programmations, efface aussi les télécommandes mémorisés.
Petits bémols: la platine est salie par le flux je suppose, Alcool à brûler avec une brosse à dent et ça brille comme un sou neuf.
Les pistes du relais sont étamées sur la photo, mais pas en réalité. Tel que, je ne pense pas que le circuit puisse supporter 10A x 230V = 2300VA, j'ai soudé 3 fils de cuivre en parallèle, et le tour est joué, ou alors étamer les pistes avec plus d'épaisseur. Ne rien faire pour des faibles puissances.
